# Stonklytics - Hacklytics'21
Stonklytics is a 3D Unity iOS mobile app, for predicting stock volatility - using data from social media. Currently, only data from [https://www.reddit.com/r/wallstreetbets/](r/wallstreetbets) is being used, the forum at the heart of the volatility that sent the stock market into a frenzy. It can easily be adjusted to account for multiple forums.

Stonklytics is a tool that can be used by all investors to track stock-related activity on this forum with 3-D visualization.

#Tech Stack
Visualization built using Unity and Blender hosted on Firebase. 
Python for web-scraping. 
IBM Watson for sentiment anaysis.
